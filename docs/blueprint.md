# **App Name**: Pet Protect Outfitters

## Core Features:

- Product Catalog Browsing: Browse surgical apparel for dogs and cats, categorized by species, size and gender (when applicable).
- Product Details: View product details including name, images, description, available sizes and a quantity field.
- Shopping Cart: Add selected products to the cart and update quantities. Allows removing items.
- Order Form: Collect customer information (name, phone number, address) before order submission.
- PDF Order Generation: Generate a PDF document of the order with customer details and product list.
- WhatsApp Order Submission: Send a pre-formatted WhatsApp message with the PDF order attached.
- Size Suggestion (AI Tool): Provide a size suggestion based on the pet's weight, leveraging AI to determine the appropriate size for the garment.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) to evoke calmness and care.
- Background color: Very light blue (#F0F8FF) for a clean and hygienic feel.
- Accent color: Light green (#BDB2FF) to complement the blue and convey health.
- Font: 'PT Sans' (sans-serif) for a modern and readable look, suitable for both headings and body text.
- Use simple and clean icons related to pets (dogs, cats, paws) to enhance navigation and product identification.
- Mobile-first, responsive layout with clear sections for product browsing, order details, and form submission.
- Subtle animations and transitions to enhance user experience, such as a smooth slide-in for the shopping cart or product details.
